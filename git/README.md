# Git tooling setup

This folder installs GitHub CLI for easier authentication and workflows.

What it gives you:
- `gh` CLI for GitHub
- browser-based login (no manual token copy/paste)

Why you need it:
- faster authentication
- fewer credential prompts
- easier repo management and PR workflows

Run:

```bash
./setup.sh
```

Note: setup may update `~/.profile` and `~/.zshrc` for PATH defaults.
