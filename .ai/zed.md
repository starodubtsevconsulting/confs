# Zed (home install)

Purpose: install the Zed editor using the official installer script.

Command:
```bash
./zed/setup.sh
```

Behavior:
- Runs the upstream install script from zed.dev.
- Ensures `~/.local/bin` is on PATH for `zed`.
- Prints the module README at the end.

Usage:
- Launch from terminal: `zed`
- Reinstall/update: rerun `./zed/setup.sh`
